# CSIT321-SlotShare
SlotShare is a community-driven platform designed to solve the urban parking crisis in Cebu City. By connecting property owners with unused driveways or lots to commuters and workers in need of secure parking, SlotShare creates a more organized and accessible city.
Features
Real-Time Availability: Instantly browse and book available parking spots in key areas like IT Park, Capitol Site, and Lahug.

CCTO Compliant Pricing: Built-in safeguards ensure all listed rates align with City Ordinance 2087 (₱25-₱60/hour), maintaining fair standards for the community.

Comprehensive Management: * For Drivers: Easily manage active, upcoming, and completed bookings via a personalized dashboard.

For Hosts: List your space with ease, specifying amenities like CCTV, 24/7 security, or EV charging stations.

Trust & Verification: Features a Trust Score system and verified user badges to ensure a safe environment for both parties.

Smart Search: Filter by area, location name, or specific amenities to find the perfect spot.

Tech Stack (Frontend)
HTML5/CSS3: Semantic structure and modern styling.

JavaScript: Dynamic interactions and real-time state management.

Figma: Original UI/UX design and prototyping.
